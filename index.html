iuwpcas by mtheoryx

Wordpress plugin to integrate Indiana University CAS Authentication. Brought to you by UITS Enterprise Web Tech Services.

This is a plugin to integrate with Indiana University's Central Authentication System (CAS). The purpose is to offload the handling of users/passwords to a trusted system for authentication (are they who they say they are?) purposes, while still maintaining control for authorization (what is this person allowed to see/do?) from within the backend of the WordPress admin panel. Users are added by IU network ID, with no need to worry about choosing passwords. Additionally, the maintainer of the blog now doesn't have to handle forgotten passwords or password resets. The plugin is provided for free (libre & gratis) to the IU Community (and anyone else who uses a similar CAS system) by the Indiana University UITS Enterprise Web Tech Services team.

For more information on the difference between Authorization and Authentication, please see the IU Information Security and Policy page on the topic.

Dependencies

WordPress Version 3.1.2 or Higher. PHP version 5

Installation Instructions

Step 1 - Install WordPress

Ensure you have a current version of WordPress installed on your web server.

Step 2 - Plugins folder

Navigate to your plugins folder

cd your_wordpress_folder/wp-content/plugins

Step 3 - Download this plugin

Download the IUWpCAS plugin from github: Downloads and unzip it into your plugins folder.

Or use Git to clone the master repository:

$ git clone git@github.com:mtheoryx/iuwpcas.git

Step 4 - Activate this plugin

You now need to activate the plugin. First, go to your admin panel in WordPress

http://your_domain_name/wp-admin

Authenticate as an administrator of your site.

Go to the plugins panel of your site.

Click "Activate" next to the IU CAS Authentication Plugin.

Important!

Step 5 - Add yourself as an admin by your IU Network ID now!

This plugin uses your IU Network ID for authentication. There will no longer be an option to log into your WordPress site with anything other than your IU Network ID.

Click "Users".

Click "Add New".

Fill in your IU Network ID in the "Username" field.

Fill in your IU email address in the "Email" field.

Optionally fill out the other fields in the form.

Choose "Administrator" in the "Role" dropdown.

Click the "Add New User" button.

Step 6 - Test

Click "Log Out".

Navigate back to your admin panel

http://your_domain_name/wp-admin

If requested, authenticate via IU CAS

You should now be allowed back into your WordPress admin panel.
